3. Command Line Parameters

Opening project from command line.

You can automatically open project if you specify its name/path from command line:

java -jar MasterofSprites.jar project_file

It is sometimes useful because on most modern operating systems, then you can drag & drop project file into MasterofSprites.jar file, and Master of Sprites will automatically open your project.

Export from command line.

Master of Sprites offers option to export its projects from command line. Syntax is following:

java -jar MasterofSprites.jar -e [format] project_file [-o output_file]

Avaible export formats are:

  • Raw format - leave format argument empty and Master of Sprites will export just same raw binary data of your sprites, without any additional informations.
  • PRG file - specify load address of prg file as the format argument. Load address should be a hexadecimal number prefixed by '$' or '0x'.
  • Assembler source code - specify name of assembler and Master of Sprites will write code compatible with syntax of specified assembler. Possible Choices are: kickass, acme or tmpx.

If you do not specify output file then Master of Sprites will just change extension of your project file to raw, prg or asm.
